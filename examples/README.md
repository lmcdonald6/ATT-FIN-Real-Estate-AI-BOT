## 🚀 Quick Start

1. **Install Dependencies**
```bash
pip install -r requirements.txt
```

2. **Set Up Your API Key**
```bash
export REAL_ESTATE_AI_KEY=your_key_here
```

3. **Launch Jupyter**
```bash
jupyter lab
```
